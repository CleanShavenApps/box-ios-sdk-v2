Overview

BoxItemsRequestBuilder is an abstract base class for building API requests relating to subclasses of BoxItem. Because this class is abstract, you should not instantiate it directly. This class does not enforce its abstractness.

This class allows constructing of the HTTP body for POST and ‘PUT` requests as well as setting query string parameters on the request.
