Overview

BoxAPIRequestBuilder is an abstract base class for request builder objects. Because it is an abstract class, you should not instantiate it directly. This class enforces its abstractness with calls to the BOXAbstract macro, which throw NSAssert failures when DEBUG=1.

This class encapsulates methods for encoding the HTTP body of an API request.
