Overview

BoxUsersResourceManager allows you to access and manipulate users via the Box API. This class is a concrete subclass of BoxAPIResourceManager. This class allows you to manipulate BoxUsers referred to by their modelID, which is an NSString.

This class enables the following operations: (Besides most operations on the user himself, /users/me, all these operations requires the user to be an enterprise administrator)

  • Get a user’s information
  • Get All Users in an Enterprise
  • Create an Enterprise User
  • Update a Userâ€™s Information
  • Delete an Enterprise User

Callbacks and typedefs

This class defines the BoxUserBlock type for successful API calls that return a BoxUser object:

typedef void (^BoxUserBlock)(BoxUser *user);

Constants

This class exposes one additional constant:

  • BoxAPIUserIDMe, for referring to the current user.
