Overview

BoxItem is an abstract model class representing items on Box. An item is any object that can be a child of a BoxFolder. These are:

  • BoxFile
  • BoxFolder
  • BoxWebLink

BoxAdditions exposes the ability to grab icons for the files. These icons are pulled out of the resource bundle that you can include in your project. This category is used by the folder picker to display default icons for each file type.
