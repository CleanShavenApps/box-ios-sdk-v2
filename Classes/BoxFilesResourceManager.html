Inherits from BoxAPIResourceManager : NSObject
Declared in BoxFilesResourceManager.h
BoxFilesResourceManager.m

Overview

BoxFilesResourceManager allows you to access and manipulate files via the Box API. This class is a concrete subclass of BoxAPIResourceManager. This class allows you to manipulate BoxFiles referred to by their modelID, which is an NSString.

This class enables the following operations:

  • Get a file’s information
  • Edit a file’s information
  • Copy a file
  • Delete a file
  • Upload a file
  • Upload a new version of a file (overwrite)
  • Download a file

Callbacks and typedefs

This class defines the BoxFileBlock type for successful API calls that return a BoxFile object:

typedef void (^BoxFileBlock)(BoxFile *file);

Return Value

The canonical URL of an upload request given the components.

Discussion

Returns a canonical URL for an API upload request given its components. Box API upload URLs are made up of three components. This helper uses uploadBaseURL and uploadAPIVersion to construct the URL.

An upload URL containing all components looks like the following:

https://upload.box.com/api/2.1/files/12345/content

The components are:

  • resource: files
  • ID: 12345
  • subresource: content

A new file upload only makes use of the resource and ID parameters and looks like the following:

https://upload.box.com/api/2.1/files/content
