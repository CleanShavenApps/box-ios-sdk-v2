Overview

BoxModelComparators defines several comparators for comparing BoxModel, BoxItem, and BoxFile objects.

Comparators are lazily created and each comparator block is only created once.

Comparators are exposed as class methods. To add more comparators, you can either extend this class or define a category.

Discussion

A comparator that orders BoxFile instances by [BoxFile SHA1].

If [BoxFile SHA1] is nil for either model, the behavior of this comparator is undefined.
