Overview

BoxParallelAPIQueueManager is an implementation of the abstract class BoxAPIQueueManager. This queue manager allows many concurrent operations at a time. This means that at any given time, only many API calls may be in progress.

BoxParallelAPIQueueManager is intended to be used in conjunction with a BoxParallelOAuth2Session. Both classes assume many concurrent BoxAPIOperationss. To ensure that the tokens will not get thrashed by concurrent refreshes of the same token, BoxParallelOAuth2Session stores a set of all access tokens that have triggered a token refresh. All BoxAPIOperation instances hold a copy of the token they were signed with and pass that to the OAuth2 session when attempting a refresh. This prevents multiple refresh attempts from the same set of tokens.

BoxParallelAPIQueueManager allows 10 concurrent download operations, 10 concurrent upload operations and 1 concurrent operation for all other API calls.
