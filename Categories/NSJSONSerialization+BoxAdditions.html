Overview

The BoxAdditions category on NSJSONSerialization provides a class method for ensuring that an object extracted from a decoded JSON object is of the expected type.

This category is used by BoxModel subclasses to extract properties from [BoxModel rawResponseJSON].

ensureObjectForKey:inDictionary:hasExpectedType:nullAllowed:

Ensure that the object at key in dictionary is a member of the expected class. NSNull may be specified as an allowable value. This method may return nil if key is not present in dictionary.

+ (id)ensureObjectForKey:(NSString *)key inDictionary:(NSDictionary *)dictionary hasExpectedType:(Class)cls nullAllowed:(BOOL)nullAllowed

Parameters

key

The key to lookup in dictionary.

dictionary

A dictionary resulting from deserializing a JSON object.

cls

The expected class of the value at key.

nullAllowed

If true, NSNull is an allowable value. Property getters that pass YES for this parameter should have the return type of id.

Return Value

The object at key in dictionary if expectations are satisfied; nil otherwise.

Discussion

Ensure that the object at key in dictionary is a member of the expected class. NSNull may be specified as an allowable value. This method may return nil if key is not present in dictionary.
